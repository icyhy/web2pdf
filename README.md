### 说明
这是一个Python 自动化脚本来实现桌面微信公众号文章批量导出成pdf文件； 脚本支持输入命令来控制保存最新的多少篇文章，如果不输入则默认保存最新的10篇，如果输入3表示保存最新的3篇，如果输入all，则表示保存所有。 

### 核心步骤

1.  **环境准备**：
    *   安装 `pywinauto`：用于控制微信 PC 端窗口（点击文章、点击菜单）。
    *   使用已有的 `playwright`：用于打开链接并生成 PDF。
    *   使用 `pyperclip`：用于获取剪贴板中的链接。

2.  **微信自动化 (PyWinAuto)**：
    *   **定位窗口**：自动找到微信主窗口和公众号文章列表。
    *   **遍历文章**：计算每篇文章的点击坐标并缓存。然后点击列表中的一篇文章。
    *   **操作详情页**：
        *   在弹出的文章独立窗口中，点击右上角的“三个点”菜单。
        *   选择 **“复制链接”**（这比“在浏览器打开”更利于自动化获取 URL）。
        *   获取剪贴板内容（文章 URL）。
        *   关闭文章窗口（为处理下一篇做准备）,然后将文章列表窗体bring to front，然后从缓存中获取下一篇文章的坐标，并点击下一篇文章，如此类推。

3.  **PDF 生成 (Playwright)**：
    *   启动 Playwright 浏览器（可设置为无头模式或有头模式）。
    *   访问获取到的 URL。
    *   等待页面加载完成（确保图片和格式正确）。
    *   调用 `page.pdf()` 将网页保存为 PDF 文件至 `results` 目录。

4.  **结果验证**：
    *   脚本运行结束后，`results` 目录下将生成对应的 PDF 文件。
